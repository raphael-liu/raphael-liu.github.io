2844-生成特殊数字的最少操作

给你一个下标从 0 开始的字符串 num ，表示一个非负整数。

在一次操作中，您可以选择 num 的任意一位数字并将其删除。请注意，如果你删除 num 中的所有数字，则 num 变为 0。

返回最少需要多少次操作可以使 num 变成特殊数字。

如果整数 x 能被 25 整除，则该整数 x 被认为是特殊数字。

示例 1：

**输入：** num = "2245047"
**输出：** 2
**解释：** 删除数字 num[5] 和 num[6] ，得到数字 "22450" ，可以被 25 整除。
可以证明要使数字变成特殊数字，最少需要删除 2 位数字。

示例 2：

**输入：** num = "2908305"
**输出：** 3
**解释：** 删除 num[3]、num[4] 和 num[6] ，得到数字 "2900" ，可以被 25 整除。
可以证明要使数字变成特殊数字，最少需要删除 3 位数字。

示例 3：

**输入：** num = "10"
**输出：** 1
**解释：** 删除 num[0] ，得到数字 "0" ，可以被 25 整除。
可以证明要使数字变成特殊数字，最少需要删除 1 位数字。

提示

  • 1 <= num.length <= 100
  • num 仅由数字 '0' 到 '9' 组成
  • num 不含任何前导零

请看 视频讲解 第二题。

一个数能被 25 整除，有如下五种情况：

  • 这个数是 0。
  • 这个数的末尾是 00。
  • 这个数的末尾是 25。
  • 这个数的末尾是 50。
  • 这个数的末尾是 75。

设 num 的长度为 n。

首先，根据题目说的，我们可以把 num 中的所有数字都删除，得到 0，这需要删除 n 次。

但如果 num 中有 0，那么删除 n-1 也可以得到 0。

接下来，看示例 1。以 50 为例说明：

  1. 从右到左遍历 num，找到第一个 0。
  2. 继续向左遍历，找到第一个 5，设其下标为 i。
  3. 删除这个 5 右侧的所有非 0 数字，这样就得到了一个以 50 结尾的数字。
  4. 删除次数为 n-i-2，例如示例 1 中 5 的下标是 i=3，需要删除 7-3-2=2 次。

其余 00,25,75 的计算方式同理，取 n-i-2 的最小值作为答案。

注意：如果没有找到要找的字符，则跳过。

答疑

问：如果删除后只得到 00，产生了前导零，这就不合法了呀？

答：不可能在删除后只得到 00，因为题目保证 num 不含前导零，如果有多个 0，那么 0 的左侧必然有非 0 数字。

复杂度分析

  • 时间复杂度：\mathcal{O}(n)，其中 n 为 num 的长度。
  • 空间复杂度：\mathcal{O}(1)。仅用到若干额外变量。
